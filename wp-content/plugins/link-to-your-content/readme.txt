== Description ==

The Link to Your Content plugin gives you more control over how you link to the content on your WordPress sites. Out of the box you can link to any of the uploaded documents in your media library along with your posts and pages. Configurable options include the ability to link to custom post types, taxonomies and any of the preceeding content across a multi-site network.

== Installation ==

1. Download and extract the plugin files
2. Upload `link-to-your-content` to the `~/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. View the settings page to turn on the ability to link to custom post types, taxonomies and across a multi-site if network enabled

== Frequently Asked Questions ==

= Where is the icon to create the link? =

The icon is on the far right of the top row of the rich text editor and looks like a piece of paper with a chain link on it.

= I clicked on the icon and the popup window closed, what gives? =

In order to create the link you have to first highlight the text you want to link to your content.
